# Tech Stack

## Core (Python Backend)
- **Language**: Python 3.11+
- **LLM Orchestration**: LangChain, LangChain-Ollama
- **Vector DB**: ChromaDB
- **Embeddings**: `nomic-embed-text` (via Ollama)
- **Model**: `qwen2.5-coder:7b` (optimized for coding)
- **Parsing**: Tree-sitter (for robust AST analysis of Java code)

## Frontend / Plugins
- **VS Code**: TypeScript
- **IntelliJ**: Kotlin, Gradle

## Target Environment (Generated Code)
- **Language**: Java 17+
- **Framework**: Spring Boot 3.x
- **Testing**: JUnit 5, Mockito, AssertJ
