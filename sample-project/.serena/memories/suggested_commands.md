# Suggested Commands

The project supports both Gradle and Maven. Since wrapper scripts (`gradlew`, `mvnw`) are not currently present in the root, system commands are assumed.

## Gradle
- **Build**: `gradle build`
- **Test**: `gradle test`
- **Run**: `gradle bootRun`
- **Clean**: `gradle clean`

## Maven
- **Build**: `mvn package`
- **Test**: `mvn test`
- **Run**: `mvn spring-boot:run`
- **Clean**: `mvn clean`

## General
- **Check Java Version**: `java -version` (Should be 17+)
